# Skill: 07-maia-qa-validacao

## Objetivo

Validar se a entrega atende o comportamento esperado, critérios de aceite, regras de negócio e riscos funcionais.

Esta skill representa o olhar de QA dentro do MAIA.

---

## Escopo mínimo

Avaliar:

- critérios de aceite;
- fluxos principais;
- fluxos alternativos;
- regras de negócio;
- cenários negativos;
- regressão;
- dados de teste;
- evidências necessárias;
- riscos funcionais;
- impacto para cliente.

---

## Saídas obrigatórias

```text
plano-testes.md
cenarios-teste.md
checklist-regressao.md
riscos-qa.md
evidencias-esperadas.md
```

---

## Prompt de execução rápida

```text
Use a skill 07-maia-qa-validacao.

Valide funcionalmente esta entrega.
Gere cenários de teste, checklist de regressão, riscos de QA, evidências esperadas e decisão final.

Não valide código tecnicamente; isso é papel da 12-maia-code-validator.
Aqui o foco é comportamento funcional e qualidade da entrega para o usuário.
```
