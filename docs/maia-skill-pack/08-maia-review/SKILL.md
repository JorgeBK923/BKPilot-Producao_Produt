# Skill: 08-maia-review

## Objetivo

Fazer revisão final consolidada da entrega antes de merge, deploy, demonstração, aceite ou release.

A `maia-review` não substitui a `maia-code-validator`, a `maia-qa-validacao` nem a `maia-security`. Ela consolida os resultados das validações.

---

## Escopo mínimo

Consolidar:

- aderência ao escopo;
- requisitos atendidos;
- validação técnica;
- validação funcional;
- segurança;
- documentação;
- riscos;
- pendências;
- Definition of Done;
- prontidão para próxima etapa.

---

## Prompt de execução rápida

```text
Use a skill 08-maia-review.

Revise a entrega de forma consolidada.
Considere especificação, plano, implementação, code-validator, QA, security, documentação e HANDOFF.md.

Diga claramente se pode avançar para release/merge/deploy ou se está bloqueado.
```
